Video-Server

A simple video server, written in C based on linux's V4l2 framework.

Check project on GitHub

Simple_video_server page.

From here, source-code can be downloaded as a tar or zip file. Code can also be viewed on the git-repository as well. To clone this code, follow this:

$ cd any_directory_you_want
$ git clone https://github.com/aman2467/simple_video_server.git
